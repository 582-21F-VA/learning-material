# Learning Material

This course uses [Eloquent JavaScript][], written by Marijn Haverbeke,
as its textbook. Each class will cover more or less one chapter from the
book.

[Eloquent JavaScript]: https://eloquentjavascript.net

Another ressource we will often refer to in class is [MDN Web Docs],
which is the de facto documentation for web technologies, including CSS,
HTML, and JavaScript.

[MDN Web Docs]: https://developer.mozilla.org/en-US/
